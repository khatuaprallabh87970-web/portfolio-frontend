# Portfolio Frontend

This is my Portfolio Website frontend.

All data is fetched from my Custom CMS Backend using APIs.

---

## Pages Included

- Home
- About
- Skills
- Projects
- Blogs
- Experience
- Testimonials
- Services
- Contact

---

## Features

- Dynamic data from CMS
- Contact form connected to backend
- Simple responsive UI
- Easy navigation

---

## Technologies Used

- HTML
- JavaScript
- React (CDN)
- Tailwind CSS
- Fetch API

---

## How to Run

1. Open `index.html` using Live Server in VS Code  
2. Make sure backend is running on:

https://cms-backend-po4n.onrender.com
